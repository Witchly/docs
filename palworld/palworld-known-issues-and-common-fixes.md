#### Server Not in the List?

If your server is playing hide-and-seek on the community list, check these settings:

* Set a Unique Name: Give your server a specific name in your settings file so it's searchable.
* Search the Right Tab: Make sure you are looking in the Community Servers list, not Official.
* Check Crossplay: Ensure all crossplay platforms are enabled in your configuration so players on different devices can find you.

#### Password Protected Servers

If your server is private, players must check the "Enter password" box _before_ clicking connect. If they don't, they'll be kicked with a "No password entered" error.
