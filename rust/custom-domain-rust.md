# üåê Custom Domain: Rust

#### 1. Get your Node IP

1. Go to the Witchly Panel.
2. Copy your numeric IP (e.g., `142.76.102.44`).
   * _Do not include the port (28015)._

#### 2. Create an A Record

1. Log into your Domain Host.
2. Go to DNS Management.
3. Add a new Record:
   * Type: `A`
   * Name: `rust` (or `play`).
   * IPv4 Address: Paste the IP (`142.76.102.44`).
   * Proxy/Cloud: Grey/Off (DNS Only).
4. Click Save.

#### 3. How to Connect

Wait 5-10 minutes for the internet to update.

Option A: Direct Connect (F1 Console)

1. Open Rust.
2. Press F1.
3. Type: `client.connect play.yourdomain.com:28015` _(Replace 28015 with your specific port)._

Option B: Server List Once your server is popular, it will appear in the Community list. The custom domain helps players verify it's really you!
